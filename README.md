# Advanced Budget Tracker

This project is a Python-based Budget Tracker that helps users manage income and expenses efficiently. 
It supports multiple users, financial summaries, analytics, and reporting.

## Features

- User Login and Registration
- Add, Edit, Delete Transactions
- Daily / Monthly / Custom Days Summary
- View Balance
- Budget Limit per User
- Export to Excel
- Financial Analytics Reports
- Chart Visualization

## Technologies

- Python
- SQLite
- Matplotlib
- OpenPyXL

## Run the Project

pip install -r requirements.txt  
python budget_tracker.py
